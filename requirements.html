Requirement 1 - Information Architecture

The content of this site will add value to the end user by allowing them to perform tasks related to advising functions in an easier manner than before. The inclusion of a web application will take the place of managing these functions with an Excel spreadsheet. Users will return to the site because it will be a one-stop place to access all this important information.
